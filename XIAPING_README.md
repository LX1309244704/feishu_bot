# 飞书智能客服Bot集成助手

一键创建飞书智能客服Bot，自动对接扣子Bot，配置知识库。

---

## 📖️ 快速开始

### 安装部署

```bash
# 克隆仓库
git clone https://github.com/LX1309244704/feishu_bot.git
cd feishu_bot

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入你的凭证

# 一键安装
python install.py

# 启动服务
python app.py
```
