## 🚀 快速开始

### 安装部署
```bash
# 克隆仓库
git clone https://github.com/LX1309244704/feishu_bot.git
cd feishu_bot

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env

# 一键安装
python install.py

# 启动服务
python app.py
```

### 服务启动后
访问：
- http://localhost:8080/health  - 健康检查
- http://localhost:8080/config  - 查看配置
